# ASL Detection

This project detects American Sign Language (ASL) hand signs from images and predicts the corresponding alphabet letter.

## Objective
To recognize hand sign images and classify them into ASL letters and special symbols.

## Features
- Image-based classification
- 29 classes (A-Z, SPACE, DELETE, NOTHING)
- Deep Learning using CNN

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV

## Output
Predicts the ASL sign shown in the input image.
