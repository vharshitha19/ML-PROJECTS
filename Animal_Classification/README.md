# Animal Image Classification

This project classifies animal images into different categories using deep learning and transfer learning.

## Objective
To identify the animal present in an image.

## Classes
Bear, Bird, Cat, Cow, Deer, Dog, Dolphin, Elephant, Giraffe, Horse, Kangaroo, Lion, Panda, Tiger, Zebra

## Technologies Used
- Python
- CNN
- MobileNet Transfer Learning
- TensorFlow / Keras

## Output
Predicts the animal name from the input image.
